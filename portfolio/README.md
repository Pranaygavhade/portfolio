# 🎬 Pranay Gavhade - Cinematic Portfolio

A high-end, storytelling portfolio website built with Next.js, Tailwind CSS, and Framer Motion.

## ✨ Features

- 🎨 Dark mode with neon purple/orange/gold/pink accents
- 🎭 Cinematic storytelling experience
- ✨ Smooth Framer Motion animations
- 🖱️ Custom cursor with glow effect
- 📊 Scroll progress bar
- 📱 Fully responsive design
- 🎯 Component-based architecture

## 🚀 Getting Started

### Installation

```bash
cd portfolio
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```

## 📝 Customization

### Update Personal Information

Edit the content in each component:

1. **Hero Section** (`components/Hero.tsx`): Update name and tagline
2. **Projects** (`components/Projects.tsx`): Add your real projects
3. **Contact** (`components/Contact.tsx`): Update email, GitHub, LinkedIn links
4. **Skills** (`components/Growth.tsx`): Modify skill levels and stats

### Add More Sections

Create new components in the `components/` folder and import them in `app/page.tsx`.
